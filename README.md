# 概要

React + Express + MongoDB のテンプレート

# 環境構築

1. make をインストール(既にインストール済みならスキップ)
2. make build
3. make up

# 環境

| Title         | URL                 |
| :------------ | :------------------ |
| WebSite       | http:localhost:3000 |
| API           | http:localhost      |
| Mongo Express | http:localhost:8081 |

# Bash

- docker exec -it backend bash
- docker exec -it frontend bash
